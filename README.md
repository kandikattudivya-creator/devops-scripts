# DevOps Automation Scripts

Bash scripts built during my DevOps learning journey.
5 years of L2 technical support experience — these scripts automate tasks I previously did manually.

## Scripts

### server_check.sh
Pings multiple servers and reports UP or DOWN status for each.

### log_analyzer.sh
Analyzes a log file — counts ERROR, WARNING, INFO levels and shows the most recent error.

### watch.sh
Monitors nginx every 5 minutes via cron. Auto-restarts if down and logs every restart with timestamp.

## How to run
bash server_check.sh
bash log_analyzer.sh
bash watch.sh
